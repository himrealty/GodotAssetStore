CGLive — Developer Guide

Install via ADB

Bypass Play Protect — install CGLive directly on any Android device

⚡
Prerequisite — Android Platform Tools
Download ADB from Google: developer.android.com/tools/releases/platform-tools ↗
Extract the zip and open a terminal inside the extracted folder before running any commands below.
Build the APK in Android Studio
Do this first before any install method below
1. Click Build → Build Bundle(s)/APK(s) → Build APK(s)
2. Wait for the build to complete — click "locate" in the notification
3. Your APK will be at:
APK path 
app/build/outputs/apk/debug/app-debug.apk
1
USB Install
Direct cable connection — simplest method
EASIEST ▼
1 On the test device go to Settings → About Phone
2 Tap Build Number 7 times until you see "You are now a developer"
3 Go to Settings → Developer Options and enable USB Debugging
4 Connect the device to your PC via USB cable. Tap "Allow" on the device when prompted
5 Run these commands in your terminal:
terminal 
# Verify device is detected
adb devices

# Install the APK (-r reinstalls if already installed)
adb install -r app-debug.apk
ℹ If adb devices shows nothing, try a different USB cable or install the device USB driver from the manufacturer's website.
2
WiFi Install
No cable needed after first setup
WIRELESS ▼
ℹ Android 11+ supports fully wireless pairing with no USB needed at all. Use the Wireless Debugging section below.
Android 10 and below — USB first, then WiFi
1 Enable USB Debugging (same as Method 1 steps 1–4)
2 Connect via USB first, then run:
terminal 
adb tcpip 5555
3 Find the device IP: Settings → About Phone → IP Address
4 Disconnect USB, then connect and install over WiFi:
terminal 
adb connect 192.168.1.x:5555
adb install -r app-debug.apk
Android 11+ — Fully Wireless, No USB Ever
1 Go to Settings → Developer Options → Wireless Debugging and enable it
2 Tap "Pair device with pairing code" — note the IP, port and pairing code shown
3 Run on your PC:
terminal 
# Pair using the IP:port and code shown on device
adb pair 192.168.1.x:PORT

# Then connect using the main port (shown under Wireless Debugging)
adb connect 192.168.1.x:5555

# Install
adb install -r app-debug.apk
3
Remote Testers
Send instructions to testers doing it themselves
REMOTE ▼
⚠ This requires testers to be comfortable using a terminal. For non-technical testers, consider Firebase App Distribution instead.
1 Send tester the APK file and the ADB platform tools download link
2 They enable Developer Options (tap Build Number 7 times) and turn on USB Debugging
3 Connect phone to their own PC via USB and run:
terminal 
adb install -r cglive.apk
ℹ This completely bypasses Play Protect and unknown sources warnings. No permissions dialogs, no blocks.
